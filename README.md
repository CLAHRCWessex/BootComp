Description:

When yoru simulation study has a large number of competiting system designs/scenarios.
It is difficult to manage the analysis.  The package is a 
multiple comparison tool for simulation output using bootstrapping.

Installation:

It is recommended that users install Anaconda - an environment manager
that bundles all of the data science libraries needed to run BootComp

(Possibly BootComp will run fine without installing the following envionment)
To get the correct libraries in place.


1. Windows -> Open Anaconda prompt.  Mac -> Open a terminal
2. Navigate to the BootComp directory
3. Run the following command:
    conda env create -f environment.yaml


This will fetch and install the libraries in a conda environment 'Bootstrap'

4. To activate the bootstrap enviroment run the following command:
    activate bootstrap
    

Intructions for use:

BootComp includes a jupyter notebook tutorial: BootCompTutorial.ipynb

Alternatively if your prefer a text editor see BootComp.py
